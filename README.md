# Penetration-Testing-Labs

##Cybersecurity lab portfolio focused on penetration testing, service enumeration, exploitation, and post-exploitation techniques performed in controlled environments. This repository documents hands-on assessments against intentionally vulnerable machines such as Metasploitable 2 and LazySysAdmin, covering protocols and services including FTP, SMB, SSH, HTTP, MySQL, DNS, SMTP, and Telnet. Includes methodology, tools used, commands, screenshots, privilege escalation techniques, and security findings for educational and skill-development purposes.
